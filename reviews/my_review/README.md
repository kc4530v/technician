# My Review

<p align="center"><img src="../../docs/reviews/my_review/screen.png" width="300"></p>

Reproduction of the **my_review** screen from `reviews/my_review.pdf` (same structure as
`screen_chat`). Big 4.8 score, 5 gold stars, "Excellent", a review card (SY avatar, Suresh
Yadav, job, date, 5 stars, review text) and a "View All Reviews" button. Brand purple `#6A4DBB`.

## Run
```bash
cd frontend && npm install && npx expo start   # press w for web
```
